# calc

A calculator for minimalist.

+ RPN
  + `↑` push
  + `↓` pop
  + `←` backspace
+ Planck Unit System
  + `g` gram
  + `m` meter
  + `s` second
+ Complex numbers
  + `log` logarithmic function
  + `  ` multiply
  + `/` invert
  + `1` unity
  + `exp` exponential function
  + `+` add
  + `-` negate
  + `0` zero
  + `†` conjugate
+ Logarithmic representation
  + a.b.c = a.b + 2πi × 0.c
    + 0.0.5 = πi
  + z X = exp z
    + 0 X = 1
    + 1 X = e
    + 0.0.25 X = i
    + 0.0.5 X = -1
