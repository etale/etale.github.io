# calc/infty

A calculator for minimalist at infinity.

+ RPN
  + `↑` push
  + `↓` pop
  + `←` backspace
+ Planck Unit System
  + `g` gram
  + `m` meter
  + `s` second
+ Complex numbers
  + `log` logarithmic function
  + ` ` multiply
  + `/` invert
  + `1` unity
  + `exp` exponential function
  + `+` add
  + `-` negate
  + `0` zero
  + `.` radix point
  + `†` conjugate
+ Logarithmic representation
  + a.b.c = a.b + 2πi × 0.c
    + `.``.``5``↑` =  0.0 + 2πi × 0.5 = πi
  + z X = exp z
    + `0``exp` = 1
    + `1``exp` = e
    + `.``.``2``5``exp` = i
    + `.``.``5``exp` = -1
